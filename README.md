## ✔️ Convention
> 하루 당 하나의 commit

1. 날짜로 제목 등록

```
[예시]
염정아
   ㄴ Algo
      ㄴ 230220.md
   ㄴ CS
      ㄴ 230220.md
      ㄴ 230221.md
```

2. git commit message
```
git commit -m "본인 이름: Algo 문제종류 문제이름"
git commit -m "본인 이름: CS 문제종류 문제간단설명"
```
```
[예시]
염정아: Algo SWEA 사다리 타기
염정아: CS 컴퓨터네트워크 OSI 7계층
```
